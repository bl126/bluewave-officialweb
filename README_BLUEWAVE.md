# Bluewave — Official Website (Prototype)

Design-first Next.js prototype for Bluewave — the Human Presence Layer on TON.

Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

Implemented

- Dark-first theme, glassmorphism, neon cyan accents
- `Header`, `Hero`, `Sections`, `WhitepaperOverlay`, `Footer` components
- Responsive layout and overlay dialog

Next recommended steps
